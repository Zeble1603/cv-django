Beautiful Weather

Used in this project : Python / Django / HTML5 / CSS3 / PostgreSQL / API calls / JSON

Check the GitHub repository

Why?

I started this project once I finished the function based view section of the Django bootcamp I was following. The main goal was to get some experience and knowledge on how to use API : How does it work ? How can I manipulate them ? I read some articles to get some project ideas. I chose to work on a website that allows the user to get the weather report of a bunch of cities he could select, like on the Iphone for example.

How?

This website is mainly constructed around a single function based view. The idea was first to store the url of the api as a variable. A form is here to feed our database of objects, in that case, they are cities. We then loop over all the objects we have and use their name to fill the api url with the python function format. We get a JSON object that we would store in a dictionary with all the datas we need (temperature, description and an icon representing the current weather). We then store the dictionary of each city in a list that we will use for the context of the view.
